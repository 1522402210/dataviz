    OpenDNS Data Visualization Framework

    Project maintained by Thibault Reuille Powered by OpenDNS

    Description

    OpenGraphiti is a free an open source 3D data visualization engine for data scientists. It helps you create advanced interactive visuals of your data.

    Requirements

    • Mac OSX or Linux
    • Clang++ or g++
    • Python 2.7.x

    Installation

    Clone the repository

    $ git clone https://github.com/opendns/dataviz.git --recursive

    Install required libraries (Note : They can also be found in the graphiti/Lib folder)

    $ pip install networkx
$ pip install pygeoip

    Compile

    $ cd dataviz/graphiti
$ make clean native
